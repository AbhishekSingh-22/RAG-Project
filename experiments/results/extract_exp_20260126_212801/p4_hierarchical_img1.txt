This document provides technical instructions for the initial setup and pairing process of a Panasonic Air Conditioner with the MirAle smart home application.

---

### 1. What is the PURPOSE of this screen/diagram?

The purpose of this guide is to provide a step-by-step visual instruction set for the user to successfully initiate the **device registration and pairing process** for a new Panasonic Non-Matter Air Conditioner within the MirAle application. It directs the user from the main app dashboard to the specific device selection menu.

### 2. What ACTION should the user take?

The user must take the following specific actions in sequence:

1.  **Initiate Pairing:** Click on the **"+"** icon located on the main app dashboard (Step 1).
2.  **Verify Prerequisites:** Before proceeding, the user must ensure that **Wi-Fi** and **Location services** (GPS/Location Access) are actively enabled on their smartphone.
3.  **Select Device Type:** Navigate to the "Select device" screen and tap on the option labeled **"Air Conditioner"** under the "Appliances" list (Step 2).
