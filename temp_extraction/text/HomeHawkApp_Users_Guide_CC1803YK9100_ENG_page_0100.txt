Filters
Filters can be used to specify the type recordings that you want to display. A single or multiple categories
can be selected. The following filter categories can be selected.
– [Person]: Displays recordings that contain detections for the motion of people.
– [Motion]: Displays recordings recorded when motion for objects was detected.
– [Manual]: Displays recordings that were recorded manually.
– [Protected]: Displays recordings that are protected.
